# We Pull at Two Homepage

Official homepage of the World of Warcraft guild **We Pull at Two** (EU - Blackrock).

## Project Vision

The goal of this project is to build a modern, fast and responsive guild homepage from scratch.

The homepage will replace the existing Google Sites page in the future and provide a better experience for members and applicants.

## Planned Pages

- 🏠 Home
- 👥 About Us
- ⚔ Raid
- 📊 Dashboard
- 📝 Recruitment
- 📰 Newsletter
